A modern, responsive React application that allows users to log expenses, categorize spending, and view real-time currency conversions using a public API.

## ✨ Features
- **Expense Management:** Add, view, and delete expenses with ease.
- **Auto-Calculations:** Real-time updates for total spending and category-wise breakdowns.
- **Currency Conversion:** Live exchange rates (USD to EUR, GBP, INR, JPY) powered by the Frankfurter API.
- **Responsive Design:** Optimized for desktop (1600px) and mobile (414px) views.
- **Error Handling:** Graceful handling of API loading and failure states.

## 🛠️ Tech Stack
- **Framework:** [React.js](https://reactjs.org/) (Vite)
- **State Management:** React Hooks (`useState`, `useEffect`)
- **Styling:** Plain CSS (Mobile-first Grid & Flexbox)
- **API:** [Frankfurter API](https://www.frankfurter.app/)
